## 🔬 PROJECT DETAILS

**Dataset:** CIFAR-10 (10,000 images, 10 categories)
**Classes:** airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck

**Features Used:**
- HOG (Histogram of Oriented Gradients) - captures shapes
- Color Histograms - captures colors
- PCA reduced 1860 features → 100 features

**Models Compared:**
- SVM with RBF kernel (WINNER: 57.6%)
- SVM with Linear kernel (49.3%)
- Logistic Regression (49.6%)
- Random Forest (45.7%)
- K-Nearest Neighbors (45.7%)
- Decision Tree (25.7%)
